## Getting Started (Linux)

### Install RabbitMQ Server

```bash
sudo apt-get install rabbitmq-server
```

### Enable RabbitMQ Web MQTT Plugin

```bash
sudo rabbitmq-plugins enable rabbitmq_web_mqtt
```

### Install dependencies

```bash
npm install
```

or with bun:

```bash
bun install
```

### Run the dev server

```bash
npm run dev
```

or with bun:

```bash
bun run dev
```
